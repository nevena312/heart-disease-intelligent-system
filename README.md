# Heart Disease Intelligent System

## Overview

Heart Disease Intelligent System is a full-stack web application that assists healthcare professionals in assessing the risk of heart disease using a machine learning model.

The system allows medical personnel to enter patient health parameters, receive a risk assessment generated by a trained neural network model, review previous assessments, analyze statistics, and add professional notes to each assessment.

The application combines a React frontend, Spring Boot backend, PostgreSQL database, and a Python AI service hosting a trained MLP model.

---

## Features

### Risk Assessment

* Input of 13 medical parameters
* Validation of all input values
* Automatic risk prediction using a trained MLP model
* Risk categorization:

  * Low Risk
  * Medium Risk
  * High Risk
* Probability score for each prediction

### Assessment History

* View all previously saved assessments
* Expand assessment details
* Display all patient input parameters
* Display prediction results
* Add and save doctor notes

### Filtering and Sorting

* Filter assessments by risk level
* Filter assessments by date range
* Sort by:

  * Newest first
  * Oldest first
  * Highest probability
  * Lowest probability

### Statistics Dashboard

* Total number of assessments
* Number of Low Risk assessments
* Number of Medium Risk assessments
* Number of High Risk assessments
* Average prediction probability
* Risk distribution chart
* Latest assessment overview

## Machine Learning Model

The system uses a Multi-Layer Perceptron (MLP) neural network trained on a heart disease dataset.

### Input Features

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* Oldpeak
* Slope
* Number of Major Vessels
* Thal

### Training Pipeline

1. Data preprocessing
2. Feature scaling
3. Class balancing using SMOTE
4. MLP model training
5. Model evaluation
6. Model deployment through FastAPI service

## API Endpoints

### Create Assessment

```http
POST /api/assessments
```

Creates a new heart disease risk assessment.

### Get Assessment History

```http
GET /api/assessments
```

Returns all saved assessments.

### Update Doctor Note

```http
PATCH /api/assessments/{id}/note
```

Updates the doctor note for an existing assessment.

## Running the Project

### 1. Start PostgreSQL

Create a database:

```sql
CREATE DATABASE heart_disease_db;
```

---

### 2. Start AI Service

Navigate to:

```bash
ai-service
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```

---

### 3. Start Backend

Navigate to:

```bash
backend
```

Run:

```bash
mvn spring-boot:run
```

Flyway migrations will be applied automatically.

---

### 4. Start Frontend

Navigate to:

```bash
frontend
```

Install dependencies:

```bash
npm install
```

Run:

```bash
npm run dev
```

Frontend will be available at:

```text
http://localhost:5173
```

## Disclaimer

This application is intended for educational and research purposes only.
